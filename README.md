# Aurora CRM — LP v2

Landing page editorial inspirada em [hubtown.co.in](https://hubtown.co.in/).

## Estrutura
- **Preloader** com counter 0→100%
- **Intro hero** com 4 stat blocks scattered
- **Words marquee** infinito
- **Scroll horizontal** com 6 capítulos pinados
- **Nav prev/next** fixa durante horizontal
- **Final CTA** editorial com form

## Stack
- HTML + CSS + JS vanilla (zero build)
- Lenis (smooth scroll) + GSAP + ScrollTrigger (CDN)
- Cormorant Garamond + DM Sans

## Como usar
Abra `index.html` no browser. Pronto.
